Delivery Exception Management Dashboard
Project Overview

The Delivery Exception Management Dashboard is a web-based internal operations tool designed to help logistics teams record, track, filter, and resolve delivery-related issues efficiently.

The dashboard enables operations managers to manage delivery exceptions through a clean and professional interface without requiring any backend services.

Features
Delivery Issue Reporting
Add new delivery exceptions
Required field validation
Issue categorization
Priority selection
Exception Dashboard
Dynamic issue table
Instant updates without page reload
Real-time status tracking
Issue Management
Mark issues as resolved
Disable resolved actions
Delete issues with confirmation prompt
Filtering System
Filter by Issue Type
Filter by Status
Dynamic row visibility using DOM manipulation
Analytics
Open Issues Counter
Resolved Issues Counter
High Priority Issue Highlighting
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
DOM Manipulation
